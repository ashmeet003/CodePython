****************    Python Modules  ***************************

-A module file contains functions, classes, and Variables.
- a regular .py file which can be imported in other files too.

functions/variables in imported module needs to called using dot operator

import myModule
myModule.greet()
myModule.x

- 'from' keyword could be used to import specific functionality from the code module
    Through this method=> dot is not needed
    However, if same name is used elsewhere in the code, namesapce conflict could occur

    from myModule import greet, x
    greet()
    print(x)

- '_name_' keyword stores name for imported module.
    During codes execution this _name_ chages to _main_

    from my_module import greet, PI

    greet("Bob")
    print(PI)# my_module.py
    def greet(name):
       print(f"Hello, {name}!")
    if __name__ == "__main__":
       # This code will only run when the module is executed as the main program
       greet("Alice")

*********
Be cautious of namespace conflict
NameError if you forgot to import module but still using its functions
import errors could occur in case of Circular Dependencies
    - which occurs due to two or more modules depend on each other

********************    Some common Modules     ***************************
math:
    math.sqrt()
    math.pi
    math.sin()

random:
    random.randint()
    random.choice()

os: # operating system related functions for file and directory
    os.getcwd() #current working directory
    os.listdir('/path') # list if files and directories in given path

datetime:
    now = datetime.datetime.now()
    now         # current date and time
    now.year    # current year
    now.strftime('%Y-%m-%d %H:%M:%S') # formatted date and time

json: # for JavaScript Object Notation
    data = '{"name": "John", "age": 30}'
    parsed_data = json.loads(data) #string turned into dictionary
    print(parsed_data['name'])   # Output: John

    person = {'name': 'Alice', 'age': 25}
    json_data = json.dumps(person) # object turned into string
    print(json_data)             # Output: {"name": "Alice", "age": 25}

urllib: # for url's and HTTP requests
import urllib.request
response = urllib.request.urlopen('https://www.example.com')
html = response.read()
print(html)                  # Output: HTML content of the webpage

dir():
    returns sorted list of object
    or with no argumented, returns list of names in current scope.
    dir([object])

    my_list = [1, 2, 3]
    print(dir(my_list))
    # returns attributes like append(), clear(), count() related to object

    import math
    print(dir(math))
    # returns sqrt(), sin(), cos(), pi....
