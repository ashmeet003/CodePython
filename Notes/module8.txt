String Slicing:
    [start:end:step-opt]
    Start in inclusive
    end-1 is inclusive
    step: in increment between steps like every other index or so
    text[1:7]
    text[::-1] # reverses the string
    text[12:5:-1] substring in reverse order

Removing Whitespaces:

    rstrip(): removes right whitespaces
    text.strip("!"): removes a character(s)
